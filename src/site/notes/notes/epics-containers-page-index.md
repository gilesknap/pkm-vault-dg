## Overview

EPICS Containers is a project to package and deploy [[EPICS\|EPICS]] IOCs as OCI containers, orchestrated via [[Kubernetes\|Kubernetes]]. It provides a standardised, reproducible way to run beamline control systems across multiple facilities.
